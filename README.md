# ⚖️ Nyaya AI – Legal Assistant for Everyone 🇮🇳

An AI-powered legal assistant that simplifies complex legal queries into easy, understandable language for everyone.

---

## 🌐 Live Demo
👉 https://nyaya-ai-legal.vercel.app

---

## 🎯 Problem Statement
Legal information is often complex, difficult to understand, and not easily accessible to common people.  
Nyaya AI bridges this gap by providing simplified legal guidance in a user-friendly way.

---

## 🚀 Features
- 🔍 Search legal topics (FIR, Divorce, RTI, Bail, etc.)
- 🌍 Multi-language support
- ⚡ Fast and responsive UI
- 🧠 AI-powered legal assistance
- 📚 Categorized legal topics for easy navigation

---

## 🛠️ Tech Stack
- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Language:** JavaScript
- **Deployment:** Vercel

---

## 🧩 How It Works
1. User enters a legal query  
2. System processes and simplifies the query  
3. Displays easy-to-understand legal information  

---

## 🏆 Use Cases
- Helps non-technical users understand laws  
- Assists students and beginners in legal learning  
- Provides quick legal guidance without complexity  

---

## 🚀 Future Enhancements
- 🎤 Voice input for queries  
- 🤖 Chatbot interaction (like ChatGPT)  
- 🌐 More Indian regional languages  
- 📄 Auto legal document generation
